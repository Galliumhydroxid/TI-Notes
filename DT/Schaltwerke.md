## RS-FlipFlop
- Set/Reset
- Wenn Reset anliegt: Ausgang auf Null
- Wenn Set anliegt: Ausgang auf Eins
- Beides anliegend nicht definiert
- Pegelgesteuert:
	- RS-Latch

## JK-FlipFlop
- Jump/Kill
- Jump -> 1
- Kill -> 0
- Jump und Kill -> Komplement
